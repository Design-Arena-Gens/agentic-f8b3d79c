## Codex · цифровой ассистент

Информационный лендинг на Next.js, который отвечает на вопрос «Привет, что ты умеешь?». Страница описывает ключевые компетенции ассистента, рабочий процесс и примеры задач, которые можно доверить ИИ.

### Функциональность

- Динамический блок «Чем могу быть полезен» с переключателями сценариев, шагами решения и метриками результата (`src/components/capabilities-rotator.tsx`).
- Герой-секция с ключевыми преимуществами и готовыми сценариями взаимодействия (`src/app/page.tsx`).
- Описание рабочего цикла и пример постановки задачи для быстрой отправки ассистенту.
- Градиентное оформление и адаптивная сетка на Tailwind CSS.

### Локальный запуск

```bash
npm install
npm run dev
```

Приложение откроется на [http://localhost:3000](http://localhost:3000).

### Сборка и проверка

```bash
npm run lint
npm run build
```
